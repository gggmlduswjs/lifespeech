## 1. 시스템 개요

## 2. 관리자 웹 (Admin)

### 2.1 대시보드

> `/dashboard`

![[screenshots/A01-대시보드.png]]

로그인 후 첫 화면. 핵심 지표를 한눈에 파악합니다:
- **상단 카드 4개**: 총 회원(1,234명), 오늘 출석(45/60), 이번 달 예약(128건), 이번 달 매출(4,250만원)
- **월별 등록 현황**: 최근 6개월 등록 추이 라인 차트
- **조치 필요**: 계약 미서명 3건, 출석 미체크 2건, 환불 대기 1건
- **오늘의 일정**: 당일 수업 시간표 (반, 강사, 인원)

좌측 사이드바에서 13개 메뉴로 이동합니다.

### 2.2 회원 관리

> 목록 → 상세 → 등록/수정

| 회원 목록 | 신규 회원 등록 |
|:---:|:---:|
| ![[screenshots/A02-회원목록.png]] | ![[screenshots/A04-회원등록.png]] |
| `/members` | `/members/new` |

**회원 목록**: 이름, 전화번호, 이메일, 등급(VIP/일반/신규 Badge), 등록일을 테이블로 표시. "신규 회원 등록" 버튼으로 등록 페이지 이동.

**신규 회원 등록**: 이름(필수), 전화번호, 이메일, 생년월일, 주소, 유입경로를 입력. 저장/취소 버튼.

**회원 상세** (`/members/:id`): 개별 회원 정보 조회.

**예약 목록**: 상태 필터(전체/대기/확정/취소) + 날짜 범위 검색. 각 예약의 확정/취소/등록 전환 액션. 우측 상단 **CSV 업로드** 버튼으로 네이버 예약 CSV를 일괄 등록.

## 5. 핵심 사용자 플로우

### Flow 1: 신규 회원 등록 (전체 여정)

```
네이버 예약 → CSV 업로드 → 예약 확정 → 회원 등록 → 수강 등록 → 계약서 발송 → 서명
```

| 단계 | 화면 | 설명 |
|------|------|------|
| 1 | A05 예약목록 | CSV 업로드로 네이버 예약 데이터 일괄 등록 |
| 2 | A05 예약목록 | 대기 상태 예약을 "확정" 처리 |
| 3 | A05 예약목록 | "등록 전환" 버튼으로 회원+수강 등록 |
| 4 | A04 회원등록 | 회원 기본 정보 입력·저장 |
| 5 | A08 등록상세 | "계약서 발송" 버튼 클릭 |
| 6 | M07 내계약 | 회원이 앱에서 계약서 확인 + 터치 서명 |
| 7 | A17 계약서상세 | 관리자가 서명 완료 확인 |

### Flow 2: 수업 당일 (강사 + 회원)

```
강사 일정 확인 → 회원 QR 체크인 → 강사 출석 관리 → 영상 업로드
```

| 단계 | 화면 | 역할 | 설명 |
|------|------|------|------|
| 1 | T01 강사홈 | 강사 | 오늘의 일정에서 수업 확인 |
| 2 | M02 QR체크인 | 회원 | 강사 QR 스캔으로 출석 |
| 3 | T02 강사출석 | 강사 | 참석/미참석 확인 + 저장 |
| 4 | A09 출석부 | 관리자 | 방문 현황 실시간 확인 |
| 5 | T06 영상업로드 | 강사 | Before/After 수업 영상 촬영·업로드 |
| 6 | M08 수업영상 | 회원 | 업로드된 영상 시청 |

### Flow 3: 매출·환불 관리

```
등록 → 매출 반영 → 환불 신청 → 승인 처리
```

| 단계 | 화면 | 설명 |
|------|------|------|
| 1 | A07 등록목록 | 수강 등록 완료 → 매출 자동 반영 |
| 2 | A24 매출 | 과정별 매출 현황 확인 |
| 3 | A25 환불 | 환불 신청 목록에서 승인/거부 처리 |
| 4 | A08 등록상세 | "환불 처리" 버튼으로 등록 취소 |

### Flow 4: 학습 콘텐츠 관리

```
관리자 콘텐츠 등록 → 회원 학습 → 진도 추적
```

| 단계 | 화면 | 역할 | 설명 |
|------|------|------|------|
| 1 | A21 콘텐츠등록 | 관리자 | 학습 자료 업로드 (카테고리, 필수 여부) |
| 2 | A20 학습콘텐츠 | 관리자 | 등록된 콘텐츠 목록 관리 |
| 3 | M05 학습목록 | 회원 | 필수 학습 목록 확인 + 재생 |
| 4 | M06 콘텐츠재생 | 회원 | 영상 시청 |
| 5 | A22 학습진도 | 관리자 | 회원별 학습 완료율 모니터링 |
