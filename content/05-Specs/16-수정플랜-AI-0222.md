## 🤖 AI 기능 상세 설계

### 1. AI 스피치 분석 (학생용)

**흐름:**
```
학생 녹음 (MediaRecorder API)
  → Blob → FormData → POST /ai/analyze-speech
  → Fastify → OpenAI Whisper API (STT → 텍스트 변환)
  → Claude API (스피치 분석 프롬프트)
  → JSON 응답 → 학생 화면 피드백 카드
```

**Claude 프롬프트 예시:**
```
다음은 스피치 학원 학생의 발화 내용입니다.
텍스트: {transcribed_text}
발화 시간: {duration}초

아래 항목을 한국어로 분석해주세요:
1. 발화 속도 (분당 단어 수, 적정 범위: 130~150 WPM)
2. 문장 구조 (명확성, 완결성)
3. 강점 1가지
4. 개선 포인트 2가지
JSON 형식으로 응답: { wpm, sentence_quality, strength, improvements }
```

**학생 화면:**
- 녹음 버튼 (3초 카운트다운 → 녹음 중 → 분석 중...)
- AI 피드백 카드 (속도 게이지 + 강점 + 개선점)

---

### 2. AI 면접 연습 (학생용)

**흐름:**
```
면접 질문 선택 (카테고리: 자기소개/지원동기/강점약점 등)
  → 학생 텍스트 답변 입력
  → POST /ai/interview
  → Claude API (면접관 역할 피드백)
  → 답변 점수 + 개선 피드백
```

**Claude 시스템 프롬프트:**
```
당신은 경험 많은 면접관입니다.
학생의 면접 답변을 평가하고 구체적인 개선 피드백을 제공해주세요.
평가 기준: 논리성, 자신감 표현, 핵심 전달력, 스피치 완성도
한국어로 응답, 200자 이내 피드백
```

---

### 3. 숙제 AI 자동 피드백

**흐름:**
```
학생 숙제 업로드 (음성 파일)
  → Supabase Storage 저장
  → 백그라운드: Whisper STT → Claude 피드백 생성
  → DB 저장 (homework_submissions.ai_feedback)
  → 강사 화면에서 AI 피드백 + 강사 피드백 함께 표시
```
