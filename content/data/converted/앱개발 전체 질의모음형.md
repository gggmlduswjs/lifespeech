구현 방법:
python
# 1단계: 금기어 리스트 만들기
금기어 = ["좀", "또", "어...", "음...", "그니까"]
# 2단계: STT 결과에서 카운트
텍스트 = "음... 저는 좀 그렇게 생각하는데요..."
금기어_횟수 = sum([텍스트.count(단어) for 단어 in 금기어])
# 3단계: 감점
점수 = 100 - (금기어_횟수 × 5)
```

> 긍정/부정 분석은 KoBERT 같은 사전학습 모델을 쓰면
> 문장 전체의 감성을 판단할 수 있어요."

### **(2) 비폭력 대화(NVC) 패턴 분석 (난이도: 최상 ⭐⭐⭐⭐⭐)**
**NVC란?**
- Marshall Rosenberg의 비폭력 대화
- 감정과 욕구를 솔직하게 표현
- 판단/평가 대신 관찰과 느낌 표현
**예시:**
| **나쁜 표현** | **좋은 표현 (NVC)** |
|:------------|:------------------|
| "당신은 이기적이에요" | "저는 외로움을 느껴요" |
| "또 늦었네" | "약속 시간이 중요해서 속상해요" |
| "이건 틀렸어요" | "제 생각엔 이렇게 하면 어떨까요?" |
**질문:**
```
"'당신은 ~다' 같은 판단 표현 대신
'저는 ~느낍니다' 같은 감정 표현을 쓰는지
자동으로 분석할 수 있나요?"
필요 기술:
고급 NLP (자연어 이해)
문장 구조 분석 (주어, 술어 파악)
감정 단어 사전 (외로움, 속상함, 기쁨 등)
커스텀 ML 모델 학습
현실적 구현:
python
# 1단계: 패턴 정의
판단_패턴 = ["~은 틀렸다", "~은 이기적이다", "당신은 ~"]
감정_패턴 = ["저는 ~느낍니다", "~때문에 속상해요", "~가 필요해요"]
# 2단계: 문장에서 패턴 매칭
텍스트 = "저는 인정받고 싶은 욕구가 있어서 외로움을 느껴요"
if any(패턴 in 텍스트 for 패턴 in 감정_패턴):
점수 += 20  # NVC 표현 사용
```
**좋은 답변:**
> "1차적으로는 키워드 매칭으로 구현하고,
> 2차적으로는 GPT API에 NVC 기준을 프롬프트로 주고
> 문장을 평가하도록 할 수 있어요."
**최고 답변:**
> "1000개 문장에 'NVC 점수'를 라벨링해서
> KoBERT를 파인튜닝하면 자동 평가 모델을 만들 수 있습니다.
> 하지만 이건 데이터 구축에 몇 달 걸려요."
```
