1. 영상 처리 & 업로드 실전 경험
"We-Grow나 OnFrame 프로젝트에서 실제로 영상 파일을 다루셨다고 하셨는데,

✅ 체크 포인트:
청크 업로드, CDN, 압축, 스트리밍 프로토콜 이해도
실제 경험 vs 이론적 지식 구분
2. React Native 앱 배포 실전 경험
"포트폴리오에 React Native 경험이 있다고 하셨는데,
실제로 앱스토어나 플레이스토어에 배포까지 완료하신 앱이 있나요?
있다면 심사 과정에서 어떤 리젝 사유를 받으셨고 어떻게 해결하셨는지,
없다면 배포를 위해 어떤 준비가 필요한지 설명해주세요."
✅ 체크 포인트:
실제 배포 경험 유무 (프로토타입 vs 실전)

3. 데이터베이스 설계 실전 경험
"학생 1만 명이 각자 50개의 강의를 듣고, 매일 진도율이 업데이트된다고 가정합니다.
이 데이터를 효율적으로 저장하고 조회하기 위한 DB 스키마를
화이트보드에 그려주시겠어요? (또는 말로 설명)
실제 프로젝트에서 DB 성능 문제를 겪어보신 적 있나요?"
✅ 체크 포인트:
테이블 설계, 인덱싱, 쿼리 최적화 이해도
실전 트러블슈팅 경험
4. API 설계 & 버전 관리 실전 경험
"포트폴리오의 FastAPI/Spring Boot 프로젝트에서
실제로 몇 명의 동시 사용자를 처리해봤나요?
그리고 API 명세서를 작성해서 프론트엔드 개발자와 협업한 경험이 있나요?
API 버전 관리는 어떻게 하셨나요?"
✅ 체크 포인트:
실제 부하 테스트 경험
협업 문서화 능력
유지보수 관점

"Quest of Seoul은 해커톤 프로젝트인데, 팀 규모와 본인의 정확한 역할을 말씀해주세요.
