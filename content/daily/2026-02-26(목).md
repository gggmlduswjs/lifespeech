# 2026-02-26 (목) — 1주차 D4

> **근무:** 10:00 ~ 19:00
> **목표:** JWT 인증 + 역할 미들웨어 (9개 시나리오 전부 통과)

---

## 작업 체크리스트

- [ ] `POST /auth/login` (bcrypt 검증 → JWT 발급)
  - Access Token: 15분, Refresh Token: 7일
  - 완료 기준: 올바른 계정 → accessToken + refreshToken
- [ ] `POST /auth/logout` (Refresh Token 무효화)
- [ ] `GET /auth/me` (토큰 검증 → 사용자 정보)
  - 완료 기준: 유효 토큰 → 200, 만료 → 401
- [ ] `POST /auth/refresh` (RT → 새 토큰 쌍)
  - 완료 기준: 만료 RT → 401
- [ ] **`requireRole` 미들웨어** (admin / instructor / member)
  - 완료 기준: member 계정으로 admin 라우트 → 403

### 테스트 시나리오 체크
- [ ] 올바른 자격증명 로그인 → 200 + 토큰
- [ ] 잘못된 비밀번호 → 401
- [ ] 유효 토큰으로 /auth/me → 200
- [ ] 만료 토큰으로 /auth/me → 401
- [ ] 토큰 없이 /auth/me → 401
- [ ] 유효 RT로 refresh → 200 + 새 토큰
- [ ] 만료 RT로 refresh → 401
- [ ] admin 토큰으로 /members → 200
- [ ] member 토큰으로 /members → 403
