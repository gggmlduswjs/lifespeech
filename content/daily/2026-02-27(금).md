## 🤖 Claude 개발 요청 순서

> 오늘은 주로 테스트/검증 위주. Claude 코드 작업이 있다면:

### 요청 1 — 버그 수정 (검증 중 발견 시)
```
[에러 메시지 전체 붙여넣기]

어떤 동작을 했을 때: [설명]
어떤 응답이 와야 하는데: [기대값]
실제로 온 응답: [실제값]

원인을 파악하고 수정해줘.
```

### 요청 2 — 시드 데이터 SQL (필요 시)
```
테스트용 시드 데이터 SQL을 작성해줘.

- 회원 5명 (members 테이블)
- 반 2개 (group 1개, 1to1 1개)
- 예약 10건 (status 다양하게: pending 4, confirmed 4, cancelled 2)

Supabase SQL Editor에서 바로 실행 가능하게.
```

### 요청 3 — Postman Collection 초안 (시간 남으면)
```
현재까지 만든 API (인증 4개 + 회원 4개 + 예약 3개)에 대한
Postman Collection JSON을 만들어줘.

환경변수: BASE_URL, ACCESS_TOKEN
각 요청에 예시 body/params 포함.
```
