> ⭐ MVP 전략: 탭 6개 중 핵심 3개만 완전 구현, 나머지 3개는 "준비 중" 표시

**기본정보 탭**
- [ ] `GET /members/:id` 연동
- [ ] 필드 인라인 수정 (클릭 → 입력 → Enter/blur → `PATCH /members/:id` 저장)
- [ ] 수정 성공 → 토스트 메시지

**수강이력 탭**
- [ ] `GET /members/:id/enrollments` 연동

- [ ] `GET /members/:id/attendances` 연동
- [ ] 회차별 타임라인 (날짜 + 상태 Badge)
