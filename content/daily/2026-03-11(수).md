## 🤖 Claude 개발 요청 순서

### 요청 1 — 회원 상세 3탭 (핵심 MVP)
```
회원 상세 페이지를 만들어줘. app/members/[id]/page.tsx

탭 구조 (6탭, 오늘은 3탭만 완전 구현, 나머지는 "준비 중"):

1. 기본정보 탭 (완전 구현)
   - GET /members/:id 연동
   - 인라인 수정: 필드 클릭 → input 전환 → blur/Enter → PATCH /members/:id 저장
   - 수정 성공: 토스트 메시지 "저장되었습니다"

2. 수강이력 탭 (완전 구현)
   - GET /members/:id/enrollments 연동
   - 목록: 반 이름 · 등록일 · 회차수 · 상태 Badge

3. 출석현황 탭 (완전 구현)
   - GET /members/:id/attendances 연동
   - 회차별 타임라인: 날짜 + 상태 Badge

4. 파일/상담/사전진단 탭: "준비 중 🔧" 텍스트만 표시

shadcn/ui Tabs, Toast 사용.
```

### 요청 2 — 수강 등록 + 출석부 그리드
```
수강 등록 폼과 출석부 그리드를 만들어줘.

1. app/enrollments/page.tsx (수강 등록)
   - 회원 검색 자동완성 (GET /members?search=...)
   - 반(클래스) 선택 드롭다운 (GET /classes)
   - POST /enrollments 연동
   - 성공: "attendances N회차 생성됨" 토스트

2. app/attendance/page.tsx (출석부)
   ⭐ MVP: 상태 3가지만 (출석/양해/결석)
   - 반 선택 탭
   - GET /classes/:id/attendance-grid 연동
   - 테이블: 행=회원이름, 열=회차번호
   - 셀 색상: 예정(gray) · 출석(green) · 결석(red) · 양해(yellow)
   - 셀 클릭 → 드롭다운 (출석/양해/결석) → PATCH /attendances/:id/status
   - 성공: 즉시 색상 변경 (optimistic update)
```

### 요청 3 — Vercel 배포 설정

Next.js 앱을 Vercel에 배포하기 위한 설정 파일을 만들어줘.

배포 후 확인 사항도 알려줘.
